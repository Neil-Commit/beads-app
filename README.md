# Beads App

**A focused, ad-free digital companion for the Holy Rosary, built for the faithful on the go.**

[Beads App Live Link](https://beads-app.vercel.app/)

## Overview

This project was born during a **Prusisyon** (Holy Week Procession). I noticed a modern dilemma: many participants were distracted by their phones, disconnected from the state of grace the moment called for. I wanted to set an example by praying the Rosary, but found that existing digital guides were either non-existent or infested with intrusive advertisements. **Beads App** is my response a tool designed to turn a digital distraction into a digital aid for the soul.

## Features

* **Mystery Detection**: Automatically suggests the mystery based on the current day of the week.
* **Progress Tracking**: Visual bead counter and percentage completion to keep your place.
* **Layered UI**: A clean, single-page application feel with modular "containers" for easy navigation.
* **Catholic News**: Integrated **CBCP RSS API** for real-time updates from the Philippines.
* **Dark Mode**: Optimized for low-light environments like evening processions or vigils.

## Technical Stack

* **HTML5**: Semantic structure with custom SVG rendering for the Rosary beads.
* **CSS3**: Custom mobile-first design with a focus on "layered" architecture.
* **Vanilla JavaScript**: Lightweight logic for all app transitions and data handling.

## Disclaimer

This app is a **personal project** and a small act of devotion. It is provided "AS IS" to assist in prayer and is **not** a replacement for a physical, blessed Rosary.
